A widget to show recently-added vehicles from WP Car Manager.

== Description ==

This widget displays the most recently-added cars (default 5, configurable) as created in WP Car Manager. The featured image, title, mileage, and price are displayed with a link to the full listing.

== Installation ==

Install from Plugins > Add New > Upload Plugin, or:

1. Upload the zip file containing this plugin to your plugins directory, and unpack it
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Activate FPCS WPCM Recent Cars.
